# 🐘 Тотем-бот Московского зоопарка

Telegram-бот-викторина «Какое у вас тотемное животное?» для популяризации программы опеки Московского зоопарка.

## ✨ Возможности

- 🎯 Викторина из 6 вопросов
- 🐾 9 животных-тотемов (манул, слон, фламинго, сурикат, белый медведь, трубкозуб, карликовый бегемот, лев, жираф)
- 📸 Реалистичные изображения животных
- 💝 Информация о программе «Возьми животное под опеку»
- 📤 Поделиться результатом в соцсетях
- 📞 Контактный механизм с зоопарком

## 🚀 Установка и запуск

### Требования
- Python 3.10 или 3.11 (Python 3.13 НЕ ПОДДЕРЖИВАЕТСЯ)
- Telegram-аккаунт
