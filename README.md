# 🐘 Тотем-бот Московского зоопарка

Telegram-бот-викторина «Какое у вас тотемное животное?» для популяризации программы опеки Московского зоопарка.

## 📖 О проекте

Бот помогает пользователям узнать своё тотемное животное через увлекательную викторину из 6 вопросов. Вопросы основаны на реальных фактах об обитателях Московского зоопарка. В конце викторины пользователь получает персонализированный результат с описанием характера животного, интересным фактом и фотографией.

**Цель проекта** — привлечь внимание взрослой аудитории к программе «Возьми животное под опеку» и рассказать о возможности помочь зоопарку.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎯 **Викторина** | 6 вопросов с вариантами ответов, система баллов |
| 🐾 **9 животных-тотемов** | Манул, слон, фламинго, сурикат, белый медведь, трубкозуб, карликовый бегемот, лев, жираф |
| 📸 **Реалистичные изображения** | Фотографии животных из Московского зоопарка |
| 💝 **Программа опеки** | Информация о том, как стать опекуном |
| 📤 **Поделиться результатом** | Копирование текста для соцсетей со ссылкой на бота |
| 📞 **Контактный механизм** | Email и телефон зоопарка, пересылка результата сотруднику |
| 🔄 **Перезапуск** | Кнопка «Пройти ещё раз» после результата |
| 💬 **Обратная связь** | Сбор отзывов пользователей для улучшения |
| 🛡️ **Безопасность** | Токен скрыт через переменные окружения |

## 🚀 Установка и запуск

### Требования

- **Python 3.10** или **3.11** (версия 3.13 НЕ ПОДДЕРЖИВАЕТСЯ)
- Telegram-аккаунт
- Git (опционально)

### Пошаговая инструкция

#### 1. Клонирование репозитория

```bash
git clone https://github.com/RyabininaTV/Totem_bot.git
cd Totem_bot
```

#### 2. Создание виртуального окружения

```bash
python -m venv .venv
```

Активация:
- Windows: .venv\Scripts\activate
- Mac/Linux: source .venv/bin/activate

#### 3. Создание виртуального окружения

```bash
pip install -r requirements.txt
```
#### 4. Создание бота в Telegram

1. Напишите @BotFather
2. Отправьте /newbot
3. Придумайте имя и username (должен заканчиваться на bot)
4. Сохраните токен

#### 5. Настройка переменных окружения

Создайте файл .env в корне проекта:

```text
BOT_TOKEN=ваш_токен_от_BotFather
```

#### 6. Настройка переменных окружения

```bash
python bot.py
```

При успешном запуске вы увидите:

```bash
INFO:aiogram.dispatcher:Start polling
```

ВНИМАНИЕ! Может понадобится VPN.

#### 7. Проверка в Telegram

Найдите своего бота и отправьте /start
